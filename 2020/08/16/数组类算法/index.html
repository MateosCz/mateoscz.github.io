寻找两个正序数组的中位数

思路一 归并

  • 错误1： 偶数个元素的数组求中位数时
int[] nums3 = [1,2,3,4]
int a = nums3[nums3.length/2];
int b = nums3[nums3.length/2-1];
double c = (a+b)/2
System.out.print(c);

里面输出结果是2.0,即使c是double类型，因为参与运算的两个数都是整型，因此会对计算结果向下取整之后再自动转型。想要让c等于2.5必须先把a和b设为double类型，才不会触发整数除法的向下取整。

11.盛水最多的容器

方法一：暴力法

时间复杂度：O（n^2）
空间复杂度：O(1)

方法二：双指针法

关键字：左右两边

模式识别：需要移动左右两头的问题可以考虑双指针

难点：如何移动指针

  • 相同情况下两边距离越远越好

  • 区域受限于较短边

    逻辑：宽度变短后要求高度增大才能保证面积增大

    做法：

    若右边比左边长，固定右指针，移动左指针

    若左边比右边长，固定左指针，移动右指针

    时间复杂度：O(n)

    空间复杂度：O(1)
